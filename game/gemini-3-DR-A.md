# 游戏策划案：Project Dr. A - V3.1

**代号**：Dr. A / 牢A博士 / 骨肉高达
**类型**：叙事模拟 / 心理恐怖 / 资源管理 / 沉浸式点击
**风格**：Lo-Fi 像素赛博朋克 + 克苏鲁式认知污染 + 故障艺术（Glitch Art）
**核心标语**：“疯子眼中的世界，比地狱更像天堂。”
**开发规模**：独立游戏（AI 辅助美术/配乐/文本）


## 一、 世界观与背景

### 1. 下城区 (The Low-Sector)

在贫富差距极端的近未来，医疗资源被垄断。你身处阴暗的下城区，这里的硬通货是器官与数据。
你是一名被吊销执照的前顶尖外科医生，代号 **Dr. A**。为了偿还天价债务，你被迫接手地下黑市的特殊订单。

### 2. 隐喻：“高达拼装” (The Assembly)

你的工作台对外宣称是**“高端仿生模型修复工作室”**。

* **表象**：你接收“板件”，打磨、组装，制作精美的 BJD 娃娃或机甲模型。
* **真相**：这些“板件”是来源非法的人体——无名尸体、基因实验失败品、异形残肢。
    你的工作是“清洗”生物特征，将其伪装成无害标本，或提取病理数据。

### 3. 核心冲突

**生存（金钱） vs. 理智（SAN值） vs. 伦理（真相）**

* **高 SAN (安全)** = 自我欺骗，视尸体为玩具，但必须靠撒谎维持。
* **低 SAN (清醒)** = 直面血腥现实，能获取高额报酬，但精神极度痛苦。
* **0 SAN (疯狂)** = 彻底异化，视罪恶为神圣，失去作为“人”的判断力。


## 二、 核心机制：三档认知滤镜

画面是主角 **Dr. A** 精神状态的投射，呈现 **U型认知曲线**。

### 第 1 档：防御机制 (The Denial)

* **SAN 值**：60% - 100%
* **视觉**：**“只是玩具”**。低保真像素，鲜艳明快（红/蓝/黄）。尸块看起来像乐高或 ABS 塑料，血液是润滑油。
* **听觉**：清脆的塑料卡扣声，舒缓的 Lo-Fi HipHop。
* **隐喻**：主角强行的心理防御，拒绝承认自己在处理尸体。

### 第 2 档：真实视界 (The Trauma)

* **SAN 值**：10% - 59%
* **视觉**：**“尸体派对”**。高精度写实像素，阴暗潮湿。苍白的骨质、氧化的脂肪、暗沉的肌肉。液体变成粘稠的血脓。
* **动态**：零件会抽搐；视野边缘出现暗角和故障噪点（Glitch）。
* **听觉**：湿润的粘滞声（咕啾）、骨骼摩擦声、微弱的呻吟、BGM 变调扭曲。
* **操作**：鼠标指针惯性漂移（模拟手抖），难度增加。

### 第 3 档：飞升异化 (The Ascension)

* **SAN 值**：0% - 9%
* **视觉**：**“不可名状的神圣”**。血肉消失，化为洁白无瑕的陶瓷、黄金或星空物质。恐怖转化为一种非人的“神性美感”。
* **听觉**：空灵的合唱，单调的耳鸣。
* **代价（Gameplay）**：
  * **认知错乱**：你会忽略警告（如警笛声、身体饥饿）。
  * **操作失控**：鼠标可能反向移动；你可能会把关键的高价零件判定为“凡俗杂质”而丢弃；你看不懂订单要求，只能凭直觉乱拼。


## 三、 玩法循环系统

界面分为：**工作台（拼装）、显示器（直播/接单）、居住舱（管理）。**

### 1. 拼装阶段 (The Work)

处理黑市订单。根据 SAN 值不同，你看到的东西不同。

* **极简模式 (Zen Mode)**：默认。
  * 有高亮吸附提示。体验 ASMR 般的解压感（虽然你在拼尸体）。
  * *缺点*：难以发现隐藏在血迹/纹身下的“病理数据”。
* **法医模式 (Forensic Mode)**：
  * 无提示。需根据骨骼断面、肌肉纹理判断位置。
  * *机制*：拼错会听到骨骼碎裂声（大幅掉 SAN）。需使用**紫外线灯**等工具检视细节。
  * *奖励*：能提取高价“数据”，解锁真结局线索。

### 2. 直播阶段 (The Stream)

通过直播控制舆论与 SAN 值。

* **表平台 (B-Tube) —— 维持谎言**
  * **策略**：打码，谎称是 3D 打印教学，聊日常。
  * **效果**：**恢复 SAN 值**（强化“这是玩具”的心理暗示），收益低，安全。
* **里平台 (Organ Hub) —— 贩卖焦虑**
  * **策略**：无码展示，特写病变部位，满足猎奇癖。
  * **效果**：**降低 SAN 值**（直面罪恶），收益极高（比特币），风险高。

#### 互动机制：【话术与故事】

* **读 SC / 放歌**：少量回复 SAN。
* **核心：【隐喻故事】**
  * 玩家根据物品描述讲述“模型背后的故事”。
  * *例子*：“这个关节很松，原来的主人喜欢荡秋千……”（暗示被捆绑悬吊）。
  * **结算逻辑**：
    * **主角**：SAN 值 **下降**（你在潜意识里承认了真相，打破了心理防御）。
    * **观众**：SAN 值 **下降**（感到细思极恐）。
    * **收益**：人气值 **暴涨**（吸引更核心的猎奇粉，为结局 C 铺路）。

### 3. 生存与伦理 (The Cost)

* **金钱**：支付房租、购买工具。
* **药物（抑制剂）**：强制将 SAN 拉回安全区（恢复玩具滤镜）。长期使用有耐药性，且价格越来越高。
* **数据交易**：将提取的“病理数据”卖给神秘组织（Scholar），换取巨额报酬，但会遭到强烈的精神冲击（SAN 值暴击）。


## 四、 物品图鉴与叙事碎片 (Lore)

故事不直接讲述，而是隐藏在**不同 SAN 值状态下的物品 Tooltip** 中。

1. **Grogg Baby (畸形标本)**
    * *[SAN > 60%]*：“定制版异形生物手办。头部造型夸张，风格独特。”
    * *[SAN < 60%]*：“人类婴儿标本。颅骨严重畸形，系母体长期吸食混合毒品所致。眼睛似乎还在盯着我看。”
    * *[SAN = 0%]*：“天使。真正的六翼天使雏形。它是为了降临而准备的容器。”

2. **14号“素体” (无皮者)**
    * *[SAN > 60%]*：“高精度红色肌肉教学模型，表面抛光完美。”
    * *[SAN < 60%]*：“死者，男，14岁。表皮被活体完整剥离。死因：剧痛导致的神经性休克。”
    * *[SAN = 0%]*：“剔除了凡俗皮囊的纯净灵魂，红宝石般的永恒艺术。”

3. **二手保温箱 (场景道具)**
    * *[SAN > 60%]*：“老旧的亚克力展示盒，有些划痕。”
    * *[SAN < 60%]*：“内壁布满了细密的血指甲抓痕。那是求生的痕迹。”
    * *[SAN = 0%]*：“神圣的摇篮。那些抓痕是经文，是它试图传达给世界的福音。”

4. **发光腺体**
    * *[SAN < 60%]*：“严重病变的甲状腺，发出幽蓝辐射光。供体长期生活在核废料区。”
    * *[SAN = 0%]*：“微缩的恒星，散发着真理之光。”


## 五、 结局设计

### 结局 A：模范技师 (The Ignorant)

* **条件**：保持高 SAN，只在表世界直播，还清债务，未触碰核心禁忌。
* **剧情**：你成功上岸，去了一家正规玩具厂。看到新闻报道黑市团伙落网，你看着手中的玩具，庆幸自己只做模型。
* **隐喻**：彻底的自我欺骗，成为了系统中快乐的盲人。

### 结局 B：飞升 (The Sacrifice)

* **条件**：长期处于 0 SAN，完成古神拼接，拒绝吃药。
* **剧情**：你不再在乎钱，认为肉体是累赘。在最后一次直播中，你对自己进行了“优化”。在观众眼中是血腥自裁，在你眼中是羽化登仙。你把自己拼成了那尊神像的最后一块拼图。
* **隐喻**：被疯狂吞噬，自我毁灭。

### 结局 C：告密者 (The Martyr) —— 真结局

* **条件**：
    1. 收集齐所有关键“病理数据”（证据）。
    2. 积累足够的粉丝量，解锁**“双平台同步推流”**技术。
    3. 在最终直播中，**主动手动移除所有滤镜**。
* **剧情**：
  * 你强行关闭了脑内的防御机制。三档画面重叠：玩具的轮廓、尸体的腐烂、神性的光辉交织在一起。
  * 你向全网展示了那些“模型”的真实面目。世界哗然，直播被掐断，破门声响起。
  * **尾声**：一个月后，一个新的大 V 直播间。主播拿出一对精美的**义眼组件**：“这是稀有的 Dr. A 系列……”
* **隐喻**：揭露黑暗的人，最终成为了黑暗本身的高价商品。


## 六、 声音与恐怖氛围

* **Jump Scare 策略**：**认知失调**。
  * 当你习惯了高 SAN 下的玩具质感时，画面突然闪烁一帧低 SAN 的真实画面（如眼球转动、鲜血喷溅），然后瞬间恢复正常。
  * 系统提示：“检测到视觉模块故障，正在重启……”（打破第四面墙）。

* **音效设计**：
  * **表模式**：Lo-Fi Beats + 塑料清脆声。
  * **里模式**：工业噪音 + 骨肉分离的粘滞声。
  * **0 SAN**：圣咏 + 耳鸣。


## 七、 技术预留

* **Shader Graph**：通过 Shader 实现滤镜的实时无缝切换，而非简单的贴图替换。
* **LLM 接口**：
  * `Input`: 玩家选择了“讲述保温箱抓痕的故事”。
  * `System`: 观众风格为“猎奇”、“不安”。
  * `Output`: (弹幕) "这纹理太真实了，主播以前是做法医的？" / "我想买这个箱子，出价 5BTC。"
