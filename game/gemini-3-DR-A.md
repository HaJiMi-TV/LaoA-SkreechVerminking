# 游戏策划案：Project Dr. A - V3.3

- **代号**：Dr. A / 牢A博士 / 骨肉高达
- **类型**：叙事模拟 / 心理恐怖 / 资源管理 / 沉浸式点击
- **风格**：Lo-Fi 像素赛博朋克 + 克苏鲁式认知污染 + 故障艺术（Glitch Art）
- **核心标语**：“疯子眼中的世界，比地狱更像天堂”、“它们只是玩具……只要你信，它们就是玩具”
- **开发规模**：独立游戏 / 3A大作（AI 编码/美术/文本）
- **参考作品**：
  - 氛围、画风：《VA-11 HALL-A》、《断指游戏》
  - 审查与操作：《Papers, Please》
  - 诡异叙事：《Inscryption》
  - 直播系统：《主播女孩重度依赖》


## 一、 世界观与背景

### 下城区 (The Low-Sector)

在贫富差距极端的近未来，医疗资源被垄断。你身处阴暗的下城区，这里的硬通货是器官与数据。
你是一名被吊销执照的前顶尖外科医生，代号 **Dr. A**。为了偿还天价债务，你被迫接手地下黑市的特殊订单。

### 角色设定

**主角**：你（玩家自定义ID）

- **背景**：前医学院高材生/法医，因“医疗事故”（实为背锅）被吊销执照，背负巨额债务。
- **现状**：隐居在廉价公寓，靠接黑市的外包活（拼装标本）和做拼装直播维生。
- **特质**：拥有一双“看透结构”的眼睛（通灵/高感知），能看到肉体上附着的“数据幽灵”。

### 隐喻：“高达拼装” (The Assembly)

你的工作台对外宣称是**“高端仿生模型修复工作室”**。

- **表象**：你接收精密的“板件”，进行打磨、清洗、组装，制作出艺术品般的 BJD 娃娃或机甲模型。
- **真相**：这些“板件”是来源于非法渠道的人体——无名尸体、基因实验失败品、甚至是某种不可名状生物的残肢。
    你的工作是“清洗”掉它们的生物特征，将其伪装成无害的标本，或提取其中蕴含的珍贵病理数据。

### 核心冲突

**生存（金钱） vs. 理智（SAN值） vs. 伦理（真相）**

- 为了还债和交房租，为了购买保持理智需要药物，你必须挣钱。
- 钱需要完成订单或博取流量。要获得更多打赏，你必须在直播中展示更猎奇的“拼装技巧”。
- 但接触越禁忌的“零件”，你的精神越不稳定，理智崩溃得越快，甚至会招致不可名状的注视。

SAN 值（Sanity，理智值）分为三档：

- **高 SAN (安全)** = 自我欺骗，视尸体为玩具，但必须靠撒谎维持。
- **低 SAN (清醒)** = 直面血腥现实，能获取高额报酬，但精神极度痛苦。
- **0 SAN (疯狂)** = 彻底异化，视罪恶为神圣，失去作为“人”的判断力。


## 二、 核心机制：三档认知滤镜

本作的核心机制。玩家眼中的画面并非客观现实，而是主角 **Dr. A** 精神状态的投射。
随着 SAN 值（理智）的降低，玩家对“拼装素材”的认知会经历一个 **U型曲线**：从无害，到恶心，最后回归到诡异的“完美”。


### 第 1 档：正常 / 防御机制 (Normal / Denial)

- **SAN 值区间**：60% - 100%
- **视觉表现**：**“只是玩具”**。
  - **材质**：低保真像素（Low-res）。看起来像乐高、高达板件或粗糙的 3D 打印件。颜色鲜艳（红蓝黄）。
  - **细节**：模糊化处理。你看不出那是一根股骨，只觉得是一个白色的长条形零件。
  - **液体**：亮绿色的润滑油，或者纯净水的颜色。
- **听觉**：清脆的塑料碰撞声（咔哒），背景是舒缓的 Lo-Fi HipHop。
- **心理隐喻**：主角潜意识里的自我保护机制，强行欺骗自己这只是一份普通的模型工作。

### 第 2 档：低 SAN / 真实视界 (Trauma / Reality)

- **SAN 值区间**：10% - 59%
- **视觉表现**：**“尸体派对”** 阴暗、噪点、色差故障（Glitch），视野边缘出现暗角
  - **材质**：高精度写实像素（High-res）。原本的塑料感褪去，露出苍白的骨质、氧化的脂肪黄、暗沉的肌肉纹理。
  - **细节**：极度丰富且令人作呕。你能看到零件上的血管在跳动，骨头断茬处的骨髓，甚至是残留的纹身。
  - **液体**：粘稠的暗红血液、浑浊的黄色脓液。操作时会有飞溅效果。
  - **动态**：“零件”会微微抽搐。拼装时，它们仿佛在抗拒你的手。
- **听觉**：湿润的粘滞声（咕啾）、骨骼摩擦的牙酸声（嘎吱）、微弱的呻吟、背景音乐扭曲变调。
- **操作影响**：鼠标指针会有惯性漂移（模拟手抖），操作难度增加。

### 第 3 档：SAN 值归零 / 飞升异化 (Insanity / Ascension)

- **SAN 值区间**：0% - 9%
- **视觉表现**：**“不可名状的神圣”**。
  - **材质**：**超越现实的完美质感**。血肉消失了，取而代之的是洁白无瑕的陶瓷、散发着幽光的黄金、或者是某种像星空一样的半透明物质。
  - **细节**：原本残缺的肢体在此时看起来像是精密的钟表结构，或者是某种神圣的几何体。所有的恐怖都转化为了一种**极度冷静的、非人的“美感”**。
  - **液体**：变成了流动的光点或水银。
  - **幻觉**：你不再是在拼装尸体，而是在进行某种神圣的献祭仪式。背景里的出租屋变成了宏伟的殿堂或星空。
- **听觉**：噪音消失，取而代之的是空灵的合唱声或单调的耳鸣频率。
- **操作影响**：操作极度流畅，仿佛有“神”在引导你的手，操作可以变得诡异
    （如自动吸附范围极大，吸附力过强、物理惯性大）
- **代价（Gameplay）**：
  - **认知错乱**：你会忽略警告（如警笛声、身体饥饿）。
  - **操作失控**：（每一场仅选定固定的失控类型）鼠标可能反向移动；你可能会把关键的高价零件判定为“凡俗杂质”而丢弃；你看不懂订单要求（订单字体乱码），只能凭直觉乱拼。
  - 玩家可能会把模型拼成“古神”形状。此时系统判定依然算“成功”（因为客户可能就是变态，或者主角产生了幻觉认为客户很满意），但依旧会推进结局 B（飞升）的进度，或者导致第二天收到的邮件内容变得不可名状。让惩罚体现在叙事走向上，而不是直接卡关。


## 三、 玩法循环系统

游戏界面分为三个主要区域：**工作台（拼装）、显示器（直播/接单）、居住舱（休息/状态）。**

游戏设定为主角为技术宅，虽仍然有时间的流逝，但并不强制主角每天必须休息。
但的订单需要完整的一轮体验才会交付成功：**拼装 → 直播 → 休息**。
交付完一个订单后，会收客户的反馈，并可以接取到新的订单。

> “客户的反馈邮件永远是基于‘完美的标本’，这与主角在 High SAN 下看到的‘玩具’形成文字上的反差恐怖。”

### 1. 拼装阶段 (The Work)

处理黑市订单。根据 SAN 值不同，你看到的东西不同。

- **极简模式（Zen Mode）**：默认
  - **操作**：类似《装机模拟器》的简化版。有高亮轮廓提示，吸附式操作。重点在于阅读物品描述和体验氛围。
  - **体验**：解压、ASMR。配合 Lo-Fi 音乐，适合挂机。
  - **代价/缺点**：获得的信息较少，难以发现“隐藏数据”。

- **法医模式（Forensic Mode）**
  - **操作**：无提示。玩家需要根据骨骼断面的形状、肌肉纹理走向、表皮的纹身来推断拼接位置。
  - **机制**：
    - 如果不小心把“左腿骨”拼到了“右臂”上，屏幕会突然闪烁（掉 SAN），耳边出现骨骼碎裂的幻听。
    - 需要使用工具（显微镜、紫光灯）检查零件，可能发现编号、抓痕或病理特征。
    - 当光标悬停或使用“显微镜”工具时，提供文字描述或波形图对比（如：显示“骨密度：左侧偏高”）
  - **奖励**：发现隐藏剧情线索，解锁隐藏的病理数据（额外收入），大幅提升成品估值。

### 2. 直播阶段 (The Stream)

直播是你获取资金和调节 SAN 值的手段，但也决定了你的结局。

直播可选不同平台

- **表平台 (B-Tube) —— 维持谎言**
  - **内容**：讲睡前小故事，自己的经历、拼装高达的技巧，展示美化过的高达照片。
  - **互动**：AI 生成的正常观众弹幕。“手艺真好”、“这是 3D 打印的吗？”
  - **作用**：**（撒谎）恢复 SAN 值**（欺骗自己，强化“这是模型”的心理暗示），收益低，安全。
    - 通过隐喻提到一部分真相：给里平台直播引流
  - 备注：
    - SAN=0 时，不建议在表平台直播。强行直播，会因说出不可控的内容被超管警告，然后强制下线。
      收到多次警告后：直播时，互动文字选项全部变成乱码（只有一条是正常的）。选错直接导致直播间被封禁 -> 失去表平台收入来源

- **里平台 (Organ Hub / Silk Road Live) —— 贩卖焦虑**
  - **内容**：各种猎奇展示，直播处理”零件“，甚至是在第 3 档状态下的疯言疯语。
  - **互动**：
    - **猎奇者**：打赏比特币，要求看“横截面”。
    - **受害者家属**：弹幕中偶尔夹杂着求救信号（需要玩家仔细分辨）。
    - **“它”**：当 SAN 值归零时，弹幕全是不可名状的符号，或者只有你能看懂的古神低语。
  - **作用**：**降低 SAN 值**，（直面罪恶），收益极高（比特币），风险高。解锁“飞升”结局线索。

#### 互动机制：【话术与故事】

- 普通互动：少量恢复 SAN 值
  - **读SC**：“感谢XXX老板的打赏。”
  - **放歌/看视频**：在等待胶水风干时，播放一段搞笑视频或舒缓音乐。
- **核心机制：【隐喻故事】**
  - 玩家可以选择讲述关于手中零件的“小故事”（基于物品描述）。
  - 表平台只能是正常故事，或隐喻类型小故事（但偶尔会收到超管警告）；里平台则可以讲述更猎奇、更直白的故事。
  - *例子*：“这个模型的手臂关节很松（实为被长期捆绑脱臼），据说原来的主人很喜欢玩荡秋千……”
  - **效果**：
    - **主角（你）**：SAN 值**略微下降**（直面残酷现实、宣泄了秘密）。
    - **观众**：SAN 值**下降**（细思极恐）。
    - **舆论**：直播间人气波动剧烈，普通观众感到不适离开，但更容易吸引到带有【特殊标签】的猎奇狂热粉（为下次进入暗网直播积累粉丝基础）。
  - “隐喻故事”的引流时维持里平台高收益的必要手段。
- **弹幕系统**：
  - 包含预设弹幕库。根据主播状态、互动行为等从特定的库中抽取弹幕。
  - *AI 扩展接口*：根据玩家的操作和讲述的故事，实时生成观众反应（惊恐、质疑、变态的兴奋）。


### 3. 生存与伦理 (The Cost)

”用钱在物理世界支付；用SAN在精神世界支付“

- 支付房租、高利贷。
- 购买**抑制剂**（回复 SAN，但有耐药性）。
- 购买**高级工具**（如紫外线灯，用于查看隐藏在“模型”上的血迹或特殊记号）。
- **药物管理**：购买“抑制剂”可以强制提升 SAN 值，让你眼中的世界变回“玩具”。但药物昂贵且有副作用。
- **特殊的“科研数据”**：
  - 某些特定的器官（如 V1.0 提到的畸胎 Grogg Baby）蕴含极高价值的病理数据。
  - 玩家可以使用工具提取数据（**卖给特定国家背景的科研组织**），但这会伴随剧烈的精神冲击（掉 SAN 暴击）。

## 五、 结局设计

### 结局 A：模范技师 (The Good Citizen、The Ignorant)

- **条件**：全程维持高 SAN 值，只在表世界直播，还清债务。
- **描述**：你搬离了下城区，找了一份真正的玩具厂工作。某天，你看到新闻报道破获了巨大的人体贩卖团伙。你看着电视里的马赛克图片，笑着说：“还好我以前做的只是模型。”
- **真相**：彻底的自我欺骗，成为了系统中快乐的盲人。

### 结局 B：飞升 (The Sacrifice)

- **条件**：长期处于 0 SAN 状态，拒绝吃药，完成所有极端订单，完成“古神几何学”拼装。
- **描述**：你不再在乎钱。你觉得自己的身体也是不完美的“零件”。在最后一次直播中，你微笑着拿起了手术刀，对自己进行了“优化拼装”。在观众眼中是血腥的自残，在你眼中是羽化登仙。
    你意识到这些“零件”是神圣的。你将自己作为最后的零件，拼装到了那尊不可名状的“神像”上。
- **画面**：屏幕最后定格在一尊由你自己的血肉组成的、奇异而绝美的 BJD 娃娃上。
- **隐喻**：彻底的疯狂与自我毁灭。

### 结局 C：告密者 (The Martyr) - 真结局

- **条件**：
    1. 收集齐所有关键“病理数据”（证据）。
    2. 可靠的传播渠道：
      - 积累足够的粉丝量，超过病毒式传播阈值。
      - 或积累足够的金钱，在暗网购买“全平台同步推流”服务。
    3. 在最终直播中，**主动手动移除所有滤镜**。
- **剧情**：
  - 你强行关闭了脑内的防御机制。三档画面重叠：玩具的轮廓、尸体的腐烂、神性的光辉交织在一起。
  - 你向全网展示了那些“模型”的真实面目。世界哗然，直播被掐断，破门声响起。
  - **尾声**：一个月后，一个新的大 V 直播间。主播拿出一对精美的**义眼组件**：“这是稀有的 Dr. A 系列……”
- **隐喻**：你揭露了系统，系统吃掉了你，并把你变成了热销商品。


## 六、 声音与恐怖氛围

### 声音设计

不同模式的风格

- **表模式**：
  - BGM：Chillhop / Lo-Fi Beats / 蒸汽波。
  - 音效：清脆的塑料撞击声（咔哒）、喷笔的嘶嘶声、砂纸打磨的沙沙声。
- **里模式（低 SAN）**：
  - BGM：原曲变调、降速，混入低频的电流声和类似心跳的鼓点。
  - 音效：湿润的粘滞声（咕啾）、骨骼摩擦的牙酸声、打磨时仿佛在锯骨头的声音。
  - 环境音：隔壁传来的沉闷撞击声，下水道的水滴声。
- **飞升模式（0 SAN）**：
  - BGM：空灵的合唱声、单调的耳鸣频率。
  - 音效：工具声变得极其柔和，仿佛在触摸丝绸或玻璃。

音效

- **动态混音**：根据 SAN 值实时调整。
- **ASMR 元素**：强调工具与材质接触的声音，但在后期这种声音会变得粘腻恶心。
- **环境音效**：
  - **正常**：雨声、霓虹灯电流声。
  - **异常**：墙壁里传来的抓挠声（暗示隔壁也是处理间）、下水道里像婴儿哭声一样的水流声。

### 恐怖氛围塑造

**Jump Scare 策略**：**认知失调**。

- 不使用突然的贴脸尖叫。
- 当你习惯了高 SAN 下的玩具质感，在处理特定高危素材时：
  画面突然闪烁一帧低 SAN 的真实画面（如眼球转动、鲜血喷溅），然后瞬间恢复正常。
- 系统提示：“检测到视觉模块故障，正在重启……”（打破第四面墙）。


## 七、 技术规划

### 技术栈

- **Unity / Godot**：适合 2D 像素与 Shader 处理。
- **Shader Graph**：实现“认知滤镜”的实时切换（噪点、色差、贴图替换）。通过 Shader 实现滤镜的实时无缝切换，而非简单的贴图替换。
- **LLM API (可选)**：用于生成千变万化的弹幕内容，确保留存率和重玩性。

### LLM 扩展预留

游戏内的文本结构设计为输入当前环境（SAN 值，直播互动操作，隐藏状态等扽），然后获取输出，
便于未来接入 LLM 动态生成。

- **静态库（预设）**：
  - 普通弹幕：“主播手好稳”、“这个关节改得好”。
  - 猎奇弹幕：“这是几号货？”、“肉质看起来有点老”。
- **动态扩展接口**：
  - `System_Prompt`: "你是一个正在观看猎奇拼装直播的观众，不知道主播拼的是尸体，但感觉哪里不对劲，风格要带有网络黑话。"
  - `Input`: 玩家选择了讲述“保温箱抓痕”的故事。
  - `Output`: (AI生成) “主播别说了，背脊发凉”、“这模型旧化做得太真了吧，像指甲刮的...”
