# 话题整理

## 00:00 第1小时

- 美国尸体处理总览 — 以个人见闻为线索，梳理“报警/不报警”两条路径与“白尸/黑尸”法理区分 — 尸体处理, 报警, 不报警, 白尸, 黑尸
- 警察介入的条件与差异 — 并非所有社区都会出警，风险与资源约束导致选择性到场与拖延 — 出警, 社区差异, 装甲车, SWAT, 风险
- 合法公民的标准流程与影视刻板 — 影视常见“合法公民走正规流程”，现实讨论更聚焦非法移民情形 — 正规流程, 影视刻板, 合法公民
- 非法移民案件的媒体叙事 — 个别正当防卫被叙述为“本国人杀害非法移民”，舆论呈倒置化 — 媒体, 正当防卫, 舆论, 西雅图案例
- 凶手与死者身份组合及概率 — 本国公民作案约5%，非法移民针对非法移民约15%，大多难侦破 — 身份组合, 概率, 侦破率
- 贫民窟/下水道区的取证困境 — 摄像头稀缺、治安恶劣与帮派阻力使事后取证与追查难度极高 — 贫民窟, 下水道, 摄像头, 取证
- 指纹与DNA库的局限 — 无合法身份者不在数据库，物证匹配难，形成大量“无头案” — 指纹, DNA, 数据库, 无头案
- 殡葬成本与选择 — 墓地需土地税与管理费，火化与骨灰在家更常见但被部分宗教视为不体面 — 墓地税, 管理费, 火化, 骨灰, 宗教
- 家属出售遗体的现实动因 — 经济压力下，家属直接与机构交易遗体以换取现金以维持生活 — 家属, 遗体交易, 警务现场, 经济压力
- 遗体“品相”与用途分级 — 健康完整更受实验室与解剖课青睐，严重损毁者价值与用途显著下降 — 品相, 寄生虫, 解剖课, 标本
- 机构使用后的安葬 — 学校/实验室使用后保留部分做标本，其余安葬并举行简易葬礼 — 高校, 实验室, 标本, 安葬, 葬礼
- 腐败环境与虫害影响 — 潮湿码头等环境加速腐败与虫害滋生，影响取证与处置成本 — 码头, 腐败, 蛆虫, 食腐昆虫
- 监狱经济与补贴循环 — 犯人跨监狱流转“刷补贴”，轻重罪挤占资源，致轻罪难长期羁押 — 监狱经济, 补贴, 流转, 资源挤占
- 摄像头为何难以普及 — 设备易被盗坏、蒙面识别受限，投入与执法收益不成比例 — 摄像头, 被盗破坏, 蒙面, 执法成本
- 黑帮破坏遗体的后果 — 对遗体的额外破坏令其无交易价值，反增清理与处置负担 — 黑帮, 遗体破坏, 无价值, 清理负担
